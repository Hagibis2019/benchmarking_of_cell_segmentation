    四、欠分割拆分¶

    欠分割是指多个独立细胞被错误标记为同一标签。NuPatch3D 提供三种欠分割修复模式：

    • Plane：基于分割平面的手动分割；
    • Auto：基于 GMM 的自动分割；
    • Manual：手动指定细胞中心的 GMM 分割。

    三种模式共用同一组 Target Labels，并可结合 Connected Components Split 进行后处理。

    在插件面板 Cell Boundary Refine 的 Under-segmentation (Split) 区域中，通过 Mode 下拉框选择分割模式。

    图 9. Under-segmentation 面板

    标签修复后，需点击 Interaction 区域中的 Commit 按钮，或按快捷键 Shift+S，将修改结果写回全局 Labels 图层。否则，修复后的标签仅保存在当前局部编辑区域中，不会同步到全局标签图层。有关提交与保存结果的详细说明，请参阅 保存结果。

    4.1 Plane 模式（平面分割）¶

    适用于两个细胞可通过单一平面分离的情况。

    4.1.1 设置目标标签¶

    在 Target Labels 输入框中输入需要分割的标签编号，多个标签以空格分隔，例如：

    114 514

    4.1.2 创建分割平面¶

    点击 Draw a Plane。

    若当前尚未创建平面基准点，NuPatch3D 会创建 Plane Label Splitting-Points 图层（蓝色点），并自动进入点选模式。此时应切换至 2D 视图，在目标细胞附近单击一次以确定平面中心。

    完成后，NuPatch3D 会自动创建：

    • Plane Label Splitting-Shapes：分割平面及法向量；
    • Plane-Spherical-Axes：三维参考坐标轴。

    同时，X、Y、Z 和 Phi、Theta 参数会自动同步为当前平面位置和方向。

    上述辅助图层仅用于交互定位，可通过快捷键 Shift+C 一键清理。

    4.1.3 调整分割平面¶

    通过以下参数调整平面位置和方向：

    • X、Y、Z：平面中心坐标；
    • Phi、Theta：平面法向量方向。

    调整过程中，3D 视图会实时更新平面位置，直至其位于目标细胞的分界处。

    4.1.4 执行分割¶

    点击 Split by Plane，NuPatch3D会将平面两侧的体素划分为不同标签。完成后，可在 2D 或 3D 视图中检查分割结果。

    4.2 Auto 模式（GMM 自动分割）¶

    Auto 模式基于高斯混合模型（GMM）自动识别并分离发生欠分割的细胞，适用于细胞边界较为明显且无需人工指定分割位置的场景。

    4.2.1 设置算法参数¶

    • Sampling Rate：体素采样比例（默认值：0.3），用于控制参与 GMM 训练的样本数量。数值越小，运行速度越快；数值越大，结果通常越稳定。
    • Split Dist：分割阈值（默认值：10），细胞直径大于该值就执行分割，反之亦然。
    • Merge Dist：合并阈值（默认值：3），两个细胞间距离小于该值就合并，反之亦然。

    4.2.2 设置目标标签¶

    在 Target Labels 输入框中输入需要自动分割的标签编号，多个标签以空格分隔。

    若留空，则对当前局部编辑区域（Bounding Box）内的所有非零标签执行自动分割。

    图 12. Auto 模式面板

    4.2.3 执行自动分割¶

    点击 Under-segmentation Fix，NuPatch3D 将自动完成聚类分析、细胞中心优化和标签重分配，并为新生成的细胞分配唯一标签 ID，确保不与现有标签冲突。分割完成后，细胞中心会自动更新。

    4.3 Manual 模式（手动指定中心）¶

    Manual 模式适用于自动分割效果不理想的情况。用户可手动指定细胞中心位置，引导 NuPatch3D 完成分割。

    4.3.1 设置细胞中心¶

    可通过以下两种方式指定细胞中心：

    方式一：手动输入坐标

    在 Coords of Centers 输入框中输入细胞中心坐标，格式为：

    Z,Y,X

    多个中心坐标以空格分隔，例如：

    10,20,30 15,25,35

    方式二：交互式选点

    点击 Select Coordinates，NuPatch3D 会创建 GMM-Coordinates 图层，并自动进入点选模式。此时应切换至 2D 视图，在目标细胞中心位置依次点击添加标记点。

    选点完成后，NuPatch3D 会自动：

    • 将坐标同步至 Coords of Centers；
    • 识别对应标签并填入 Target Labels。

    GMM-Coordinates 仅用于辅助选点，不参与最终分割计算。完成分割后，可通过快捷键 Shift+C 删除该辅助图层。

    图 15. Manual 模式选点示例

    4.3.2 执行分割¶

    点击 Under-segmentation Fix，NuPatch3D 将以用户指定的中心坐标作为初始细胞中心，对 Target Labels 执行分割，并自动更新标签结果。

    图 16. Manual 模式分割结果示例

    4.4 连通域重标号¶

    若同一标签包含多个不连通区域，可点击 Connected Components Split 进行处理。

    NuPatch3D 会对 Target Labels 中的每个标签执行三维连通域分析，保留最大连通域的原标签，其余连通域自动分配新的标签 ID，从而解决欠分割。

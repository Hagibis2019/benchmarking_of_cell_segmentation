# 九、 恢复结果

## 9.1 撤销与重做（Undo / Redo）

点击 <kbd>Interaction</kbd> 区域中的 <kbd>Undo</kbd> 按钮，或按快捷键 <kbd>Shift</kbd>+<kbd>Z</kbd>，可撤销上一步操作，并将标签数据恢复到修改前的状态。

点击 <kbd>Interaction</kbd> 区域中的<kbd>Redo</kbd> 按钮，或按快捷键 <kbd>Shift</kbd>+<kbd>B</kbd>，可恢复刚刚撤销的操作。

NuPatch3D 支持最多 10 步撤销与重做。如果是napari的画笔、橡皮擦、填充等操作，需要<kbd>Control</kbd>+<kbd>Z</kbd>进行撤回。


## 9.2 使用 JSON 日志恢复结果

当程序异常退出、标签数据意外丢失，或需要恢复此前的修复结果时，可利用原始图像、初始预分割标签图像以及对应的 JSON 日志文件重建修复结果。

点击 <kbd>Apply JSON Log</kbd>，选择由 NuPatch3D 生成的 JSON 日志文件。NuPatch3D 将自动读取日志中的历史记录，并按照保存顺序将已提交的修改重新应用到当前标签图层。

恢复过程中，NuPatch3D 会自动检查日志文件与当前图层的兼容性。若日志中记录的图层名称与当前 napari 中的图层名称不一致，系统会尝试自动匹配合适的图层；若无法唯一确定目标图层，则会弹出对话框供用户手动选择。

恢复完成后，标签数据将恢复到日志文件记录的最新提交状态，可继续进行后续编辑或直接导出结果。

> **提示**
>
> 为确保恢复结果正确，建议使用生成该 JSON 日志时对应的原始图像和初始预分割标签图像。
